# 📁 redneckryan-talking

An OCRP-style "Talking" UI script for FiveM that displays a list of players currently speaking on your screen.

⚙️ Features
Zero Dependencies: Works standalone out of the box.

Universal Voice Support: Works with any voice system utilizing FiveM's native Mumble audio routing (including pma-voice, SaltyChat, and vanilla Mumble-VoIP).

Lightweight: Minimal performance footprint.

🚀 Installation Instructions
Download the resource and place the redneckryan-talking folder into your server's resources directory.

Open your server.cfg file.

Add the following line below your preferred voice resource (to ensure it initializes correctly after audio routing is set up):

Code snippet
ensure redneckryan-talking
